Saya berakar dari dunia infrastruktur IT—seorang *PC Builder* yang hobi mengulik *Server* dan *Networking*. Saat ini, saya sedang mengekspansi keahlian saya ke dunia **Software Development** dan **Artificial Intelligence** untuk menciptakan otomatisasi dan solusi yang menggabungkan *hardware*, jaringan, dan kode.

- 🖥️ **What I love:** Merakit PC, mengonfigurasi *Home Lab/Server*, dan mencari celah keamanan dalam infrastruktur.
